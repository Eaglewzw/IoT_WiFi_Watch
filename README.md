# IoT_WiFi_Watch

## 项目简介
基于 ESP8266 NodeMCU 的物联网 WiFi 手表，集成网络时间、天气预报、MQTT 智能家居控制和闹钟功能。本项目在开源项目 WeatherStation 基础上进行了二次开发与优化。

## 功能介绍
- **时间同步**：通过 NTP 协议获取实时网络时间（上海时区）。
- **天气预报**：调用心知天气 API，支持 39 种天气状态显示及未来三天预报。
- **智能家居**：通过 MQTT 协议远程控制台灯、空调，并实时接收室内温湿度。
- **事务闹钟**：支持三组独立闹钟，数据经 EEPROM 掉电保存。
- **配网管理**：支持 SmartConfig (ESP-Touch) 快速配网及 WiFi 自动重连。
- **系统设置**：可调节屏幕亮度、音量、息屏时间，查看电池状态。

## 状态机架构
固件采用两层状态机架构，通过 D7 按键切换 5 个主界面帧 (Frame)：

```
                         D7 按键 (切换主界面)
                    ┌──────────────────────────┐
                    ▼                          │
    ┌─────────────────────────────────────────────────────┐
    │              主状态机 (uiFrameIndex 0~4)              │
    │  ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐
    │  │ Frame0 │ │ Frame1 │ │ Frame2 │ │ Frame3 │ │ Frame4 │
    │  │ 主界面  │ │  闹钟  │ │  天气  │ │  MQTT  │ │  设置  │
    │  └────────┘ └────────┘ └────────┘ └────────┘ └────────┘
    └─────────────────────────────────────────────────────┘
```
*注：在各 Frame 内，可通过 D3/D5/D6 按键进行具体的数值选择与设置。*

## 硬件设备
- **MCU**: ESP8266 NodeMCU
- **显示屏**: 0.96 寸 SSD1306 OLED (I2C)
- **按键**: 4 个物理按键 (D3, D5, D6, D7)
- **辅助**: 蜂鸣器 (闹钟)、LED (状态提醒)
